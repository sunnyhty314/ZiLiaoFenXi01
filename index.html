项目 公式 说明
基期A2 \[ A2 = \frac{A1}{1 + r1} \] 基期A的计算公式
基期B2 \[ B2 = \frac{B1}{1 + r2} \] 基期B的计算公式
增长量x1 \[ x1 = A1 - A2 \] A的增长量
增长量x2 \[ x2 = B1 - B2 \] B的增长量
现期比重P \[ P = \frac{A1}{A1 + B1} \] 现期A在A+B中的比重
基期比重P' \[ P' = \frac{A2}{A2 + B2} \] 基期A在A+B中的比重
两期比重差d \[ d = \frac{A1}{B1} \times \frac{(r1 - r2)}{(1 + r1)} \] 现期比重与基期比重的差值
比值增长率k \[ k = \frac{(r1 - r2)}{(1 + r2)} \] 现期比重相对于基期比重的增长率
基期和S \[ S = A2 + B2 \] 基期A和基期B的和
基期差D \[ D = B2 - A2 \] 基期B与基期A的差值
隔年增长率r \[ r = r1 + r2 + r1 \times r2 \] 隔年的增长率
AB和增长率r3 \[ r3 = \frac{A1 + B1}{A2 + B2} - 1 \] 现期A+B相对于基期A+B的增长率
AB差增长率r4 \[ r4 = \frac{B1 - A1}{B2 - A2} - 1 \] 现期B-A相对于基期B-A的增长率
