项目 公式 说明
基期A1 \[ A1 = \frac{B1}{1 + r1} \] 第一个现期B1的基期
基期A2 \[ A2 = \frac{B2}{1 + r2} \] 第二个现期B2的基期
增长量Δ1 \[ \Delta 1 = B1 - A1 \] 第一个现期1的增长量
增长量Δ2 \[ \Delta 2 = B2 - A2 \] 第二个现期2的增长量
现期比重P1 \[ P1 = \frac{B1}{B1 + B2} \] 第一个现期B1在B1+B2中的比重
现期比重P2 \[ P2 = \frac{B2}{B1 + B2} \] 第二个现期B2在B1+B2中的比重
基期比重P'1 \[ P'1 = \frac{A1}{A1 + A2} \] 第一个基期A1在A1+A2中的比重
基期比重P'2 \[ P'2 = \frac{A2}{A1 + A2} \] 第二个基期A2在A1+A2中的比重
两期比重差d1 \[ d1 = \frac{B1}{B2} \times \frac{(r1 - r2)}{(1 + r1)} \] 第一个现期比重与第二个现期比重的差值
两期比重差d2 \[ d2 = \frac{B2}{B1} \times \frac{(r2 - r1)}{(1 + r2)} \] 第二个现期比重与第一个现期比重的差值
比值增长率k1 \[ k1 = \frac{(r1 - r2)}{(1 + r2)} \] 第一个现期比重相对于第二个现期比重的增长率
比值增长率k2 \[ k2 = \frac{(r2 - r1)}{(1 + r1)} \] 第二个现期比重相对于第一个现期比重的增长率
基期和S \[ S = A1 + A2 \] 基期A1和基期A2的和
基期差D \[ D = A2 - A1 \] 基期A2与基期A1的差值
隔年增长率r \[ r = r1 + r2 + r1 \times r2 \] 隔年的增长率
AB和增长率r3 \[ r3 = \frac{B1 + B2}{A1 + A2} - 1 \] 现期B1+B2相对于基期A1+A2的增长率
AB差增长率r4 \[ r4 = \frac{B2 - B1}{A2 - A1} - 1 \] 现期B2-B1相对于基期A2-A1的增长率
